Use a PowerPC Mac as your daily machine

If you have a late G4 machine or G5 machine you can still use them pretty easily. Now, I know it is possible to use even earlier machines but most of the stuff you need is only available on Leopard and you need a fast machine for that. Here is a list of apps you might want or need:

First of all, install Sorbet Leopard. It is an unofficial version of Leopard that is a lot faster and lighter (Link below, installation instructions are on the Macintosh Garden).

  • InterWebPPC - Web browser based on Firefox
  • Leopard Webkit - Faster than InterWeb but can open less sites as it is a bit older
  • TenFiveTube - YouTube Client with plugin support
  • For TenFiveTube: Make sure to install the ViewTube and ViewTube+ plugins for smoother experience (Check description of the source page). Also change your user agent to iPad.
  • DiscordPPC - Discord Client that uses the website version (doesn't work anymore)
  • Discord Lite - Lighter Discord Client made by dosdude1
  • Spotfy - Spotify alternative that takes music from Youtube
  • PPCMediaCenter - allows you to watch YouTube videos & streams + Twitch (doesn't work without patching)
  • Transmission - torrent client

You can watch 720p and 1080p videos on late G4s and G5 Macs in fullscreen mostly smoothly by downloading the desired video and then using HandBrake to convert it to "Web optimized" and to use "MPEG-4 (ffmpeg)" as the video codec. After that just open it with CorePlayer and it should work. It should give a boost to older machines that can't do 360p normally but don't expect too much."
